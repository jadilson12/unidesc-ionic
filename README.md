## Descrição

Projeto Ionic com angular

## Requisito funcional

- crud aluno
- crud professor
- crud turma

## Requerimento nescessário

- Node 10.x
- NPM 6.x
- ionic 4.x

## Instalação do cli do ionic

```bash
$ npm install -g @ionic/cli
```

## Baixar o projeto

```bash
$ git clone https://github.com/jadilson12/unidesc-ionic
$ cd unidesc-ionic
$ npm install
```

## Iniciar o projeto

```bash
$ ionic serve
```

## Url para visualização

```bash
$ http://localhost:4200/
```
